# 编码

三种帧

1. I帧（关键帧）
2. P帧（前向预测编码帧）
3. B帧（双向预测内插编码帧）

NALU：网络提取层单元，二进制流

一个视频，拆分成N个帧，每一个帧拆成N个片，每一个片都放在一个NALU里面，最终形成一个NALU序列

# RTMP

推流：将二进制流打包成网络包进行发送，使用RTMP协议，基于TCP

重要信息：

1. 版本号
2. 时间戳

拉流：同样使用RTMP协议
